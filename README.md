## Java Bank Account Management System

## 📌 Description
This is a simple Java-based console application that demonstrates Object-Oriented Programming (OOP) concepts. The system allows users to create and manage multiple bank accounts using a menu-driven interface.

## 🚀 Features
- Create multiple bank accounts
- Deposit money
- Withdraw money
- Display account details
- Menu-driven user interaction

## 🧠 Concepts Used
- Classes and Objects
- Encapsulation
- ArrayList (Collections Framework)
- Conditional Statements
- Loops

## ⚙️ Technologies
- Java (JDK 17)

## ▶️ How to Run

1. Compile the program:
   javac Main.java
2.Run the program:
  java Main
